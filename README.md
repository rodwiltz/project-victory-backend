# 🚀 Project Victory Backend

Backend services for **Project Victory**, the operating platform for **Big W Events**.

## Purpose

Project Victory is designed to automate and standardize the rental lifecycle for an event rental business.

Core capabilities include:

* Inventory Management
* Rental Management
* QR Code Tracking
* Checkout & Returns
* Requirement Verification
* Inventory History
* Operational Controls

## Architecture

The backend is built with Google Apps Script using a modular service-oriented architecture.

### Core Services

* AgreementService
* FulfillmentService
* HistoryService
* InventoryService
* RentalCartService
* RentalService
* RequirementService
* ReturnService
* SettingsService
* Utilities
* ValidationService
